# Papo de Homem para Homem - Website

## Sobre

Sabemos que a violência doméstica é um problema de todos. Este projeto foi criado pela Polícia Judiciária Civil de Mato Grosso para abrir um diálogo honesto entre homens. Acreditamos que a mudança começa com a informação e com a coragem de assumir a responsabilidade. **Junte-se a nós nesta causa.**

## Participação da Polícia Civil

**Projeto apresentado:** "Papo De Homem Para Homem".       
**Ministrante:** Policial Civil Nilton Cesar Cardoso.       

## Objetivos do Projeto

- Conscientizar e mitigar a ignorância sobre a Lei Maria da Penha.
- Sensibilizar o público masculino sobre o fim da violência contra a mulher.

Nossa missão é transmitir essas mensagens da melhor forma possível.
